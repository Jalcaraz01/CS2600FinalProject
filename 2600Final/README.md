# C Project: Who wants to be a millionaire?
Developed by: Justin Alcaraz, Steve Chu, Giselle Ng, Ayla Santos

The purpose of this project was to create a playable recreation of the popular  game show "Who wants to be a millionaire?" in C. As on tv, players of this game are asked 15 questions of increasing difficulty to win cash prizes, and possibly one million dollars if all 15 questions are answered correctly!

# HOW TO PLAY
This game runs in a terminal window, to  start it follow these steps:
1) Download main.c, questions.txt, and makefile into one folder
2) In terminal, change directory to the folder the files are in
3) Type and enter the command, "make"
4) Type and enter "./game" to start the game!

Once the game has started,
Enter A, B, C, or D to answer questions
Enter F to use one of two available 50/50 lifelines (reveals two options, one of which is the correct one)
Enter H to use one of two available "Ask Host" lifelines (reveals the correct answer)
Enter W to walk away with your current winnings
